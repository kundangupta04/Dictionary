# Dictionary Web Application

A responsive web application where users can search for an English word and view:

- Meaning  
- Part of speech  
- Example sentence  
- Phonetic pronunciation  
- Audio pronunciation (if available)

Built using **HTML, CSS, and JavaScript**, powered by the Free Dictionary API.

## Features

- Search by clicking button or pressing Enter  
- Shows valid meaning and part of speech  
- Example sentence or fallback text  
- Phonetic text display  
- Play audio pronunciation when available

- Error handling for invalid inputs and API failures  
- Clicking the logo refreshes the app  
- Clean, modern, responsive UI

## API

https://api.dictionaryapi.dev/api/v2/entries/en/<word>

## How to Run

Download the project and open `index.html` in your browser.

No server needed.
